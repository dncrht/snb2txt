# Snb2txt

snb2txt is a simple CLI tool to extract text from Samsung Notes files (.snb).

Install via:
```bash
$ gem install snb2txt
```


## Usage

As any other CLI command:

```bash
snb2txt your_note_file.snb
```

The text contained in the note will appear on the standard output.
